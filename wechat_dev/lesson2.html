快速编写代码

创建文件，起名叫 www.js。源代码如下：

const express = require('express');
const app = express();
const http = require('http');
const server = http.createServer(app);
const PORT = 8080;

server.listen(PORT, () => {
  console.log(`正在监听${PORT}端口...`);
});

app.get('/', (req, res) => {
  res.send('Hello World');
});

启动WEB服务

打开命令行终端，进入项目根目录。执行node www.js命令后，控制台输出“正在监听8080端口...”，表示WEB服务已经成功启动。

内网穿透

从 Ngrok (opens new window) 下载适合自己操作系统的 ngrok 版本。打开命令行终端，找到 ngrok 所在的位置，执行如下命令：

./ngrok http 8080

在浏览器中打开终端中给出的外网网址，页面显示“Hello World”，表示外网用户已经可以访问我们本地的WEB服务了。
